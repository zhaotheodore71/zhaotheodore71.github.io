# 俄罗斯方块游戏

这是一个使用HTML、JavaScript和Tailwind CSS实现的经典俄罗斯方块游戏。游戏具有完整的功能，包括方块移动、旋转、消除行、得分系统、等级提升等，同时支持键盘和触摸屏控制。

## 游戏特性

- 🎮 **经典玩法**：完整实现了俄罗斯方块的经典游戏机制
- 🎨 **现代UI**：使用Tailwind CSS构建的美观界面
- 📱 **响应式设计**：适配桌面和移动设备
- ✨ **视觉效果**：包含方块阴影、消除行动画等效果
- 🎯 **得分系统**：根据消除行数和等级计算得分
- 🔄 **等级提升**：随着消除行数增加，游戏难度逐渐提高
- 🎯 **触摸屏支持**：移动设备可通过触摸操作游戏

## 操作说明

### 键盘控制
- **← →** - 左右移动方块
- **↑** - 旋转方块
- **↓** - 加速下落
- **空格** - 直接落到底部
- **P** - 暂停/继续游戏

### 触摸屏控制
- **左右滑动** - 移动方块
- **向下滑动** - 加速下落
- **向上滑动** - 直接落到底部
- **轻触** - 旋转方块

## 如何运行游戏

### 方法一：直接打开HTML文件

1. 找到 `index.html` 文件
2. 双击打开，或右键选择使用浏览器打开

### 方法二：使用HTTP服务器

如果直接打开HTML文件出现跨域问题，可以使用HTTP服务器：

```bash
# 使用Python启动HTTP服务器
python -m http.server 8000

# 或使用Node.js（需安装http-server）
npm install -g http-server
http-server -p 8000
```

然后在浏览器中访问 `http://localhost:8000`

## 游戏机制

1. **方块类型**：游戏包含7种不同形状的方块（I、J、L、O、S、T、Z）
2. **得分规则**：
   - 软下落（↓键）：每格1分
   - 硬下落（空格键）：每格1分
   - 消除1行：100分 × 当前等级
   - 消除2行：300分 × 当前等级
   - 消除3行：500分 × 当前等级
   - 消除4行：800分 × 当前等级
3. **等级提升**：每消除10行，等级提升1级，下落速度增加

## 开发说明

### 主要文件

- `index.html` - 游戏界面和样式
- `game.js` - 游戏逻辑实现

### 游戏配置

游戏的主要配置参数在 `game.js` 文件顶部定义：

- `COLS` - 游戏板列数（默认10）
- `ROWS` - 游戏板行数（默认20）
- `BLOCK_SIZE` - 方块大小（默认30像素）
- `dropInterval` - 初始下落速度（默认1000毫秒）
